# Batch 1: Build System Unification & Deploy Infrastructure

## Context

The OCWS project has 20 GUI apps, 21 CLI utils, and 7 daemons. After investigation, **all targets are already in `build.zig`** (the audit's "15 missing" finding was stale). The real gaps are:

1. **`ocws.zig` dispatch table** only knows 17 subcommands — missing 19 GUI apps
2. **`ocws.zig` binary lookup** hardcodes `zig-out/bin/` — won't find installed binaries
3. **`install.sh` never deploys `.desktop` files** — no app appears in app menus
4. **`install.sh` has duplicate `*)` wildcard** — dead code on line 101
5. **`shell-mode.sh` only knows 4 of 13 modes** — rejects valid modes
6. **Zig version mismatch** — ci.yml uses 0.16.0, release.yml uses 0.13.0

## Changes

### 1. `src/ocws.zig` — Add missing subcommands + PATH lookup

Add 19 missing GUI app entries to the `subcommands` array:
```
fonts-mgr, dock-mgr, dotdesktop-mgr, pkgmgr, llm-runner,
welcome, theme-center, workspace-mgr, equalizer,
equalizer-gl, equalizer-qs, waveform-gl, waveform-qs,
speaker-gl, speaker-qs, snake-game, todomvc, datetime, wallpaper-picker
```

Fix `execExternal()` to search multiple paths instead of hardcoding `zig-out/bin/`:
1. First try `zig-out/bin/ocws-{name}` (dev build)
2. Then try `{HOME}/.local/bin/ocws-{name}` (installed)
3. Then try `ocws-{name}` via `execvp` (PATH lookup)

Update `printGodStatus()` and `runGodInstall()` to iterate all subcommands (including GUI apps).

### 2. `install.sh` — Deploy .desktop files + fix duplicate wildcard

Add a deployment step after binary installation:
```bash
# Deploy .desktop files to ~/.local/share/applications/
mkdir -p ~/.local/share/applications
for desktop in "$SCRIPT_DIR"/dotfiles/applications/*.desktop; do
    [ -f "$desktop" ] && cp "$desktop" ~/.local/share/applications/
done
```

Remove the duplicate `*)` wildcard on line 101 (dead code — the second `*)` can never be reached).

### 3. `scripts/actions/shell-mode.sh` — Support all 13 modes

Update the validation check (line 18) to accept all modes from `install.sh`:
```
doublepanel, dms, noctalia, minimal, tworow,
lxqt-bottom, lxqt-classic, lxqt-minimal, lxqt-standalone,
lxqt-dual-lxqt, lxqt-vertical, zigshell-cairo-pango, zigshell-blend2d
```

Update the help text (lines 9-14) to list all modes.

### 4. `.github/workflows/release.yml` — Align Zig version

Change `version: 0.13.0` to `version: 0.16.0` to match `ci.yml`.
